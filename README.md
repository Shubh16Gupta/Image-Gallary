# Image-Gallary
<h1>🖼️ Responsive Image Gallery Website</h1>
This is a fully responsive Image Gallery website created using HTML and CSS. It features a modern UI, grayscale-to-color image hover effects, and smooth transitions. This project is perfect for showcasing frontend skills and creating a visually appealing portfolio piece.
<br><hr>
<h2>🚀 Features</h2>
🎨 Clean and modern design
<br>
💻 Fully responsive layout using Flexbox
<br>
🖼️ Image cards with hover animations (grayscale to color)
<br>
✨ Smooth transition effects
<br>
📱 Mobile-friendly view for all screen sizes
